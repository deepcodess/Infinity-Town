Infinity Town is a real-time WebGL 3D city simulation built using the Three.js framework. The project recreates a procedurally generated city with cinematic movement, simulation-based traffic, dynamic lighting and evening environment ambience.

The project draws inspiration from the original experimental build INFINITOWN and the later visual concept INFINITY TOWN, adding custom asset development, modern rendering pipeline and performance-focused scene optimisation.
